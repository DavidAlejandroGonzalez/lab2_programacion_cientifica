# lab2_programacion_cientifica
El objetivo principal es aplicar funciones avanzadas de Python para optimizar el procesamiento y análisis de grandes volúmenes de datos. Se trabajó específicamente con:Funciones lambda para operaciones anónimas.map() para transformación de datasets.filter() para limpieza de datos.reduce() para agregación de datos.Diseño de pipelines de procesamiento funcional. Contenido del Repositoriolab2.ipynb: Notebook de Jupyter con la resolución de las 6 actividades y el desafío avanzado.README.md: Documentación del proyecto. Actividades DesarrolladasUso de lambda: Creación de funciones matemáticas personalizadas.Transformación con map(): Conversión de unidades de medida.Filtrado con filter(): Limpieza de errores en datasets de temperatura.Agregación con reduce(): Operaciones acumulativas de datos.Pipeline de Procesamiento: Integración de múltiples funciones en un solo flujo.Simulación de Big Data: Procesamiento eficiente de 1.000.000 de registros.Desafío Avanzado: Simulación de monitoreo de sensores industriales en tiempo real.
